# Obscura - Vintage Camera Rental Landing Page

A modern, premium startup landing page for Obscura, a smart self-service vintage camera rental platform in Jakarta.

## 🎬 Project Overview

Obscura is a cutting-edge platform that brings back the magic of vintage photography by making Instax and analog cameras available through automated rental stations in urban areas. Users can locate machines, rent cameras, explore the city while capturing memories, and receive their deposit refund automatically.

## ✨ Features

- **Modern Design**: Premium startup aesthetic with vintage photography inspiration
- **Responsive**: Fully optimized for mobile, tablet, and desktop
- **Smooth Animations**: Framer Motion animations for engaging user experience
- **Component-Based**: Reusable React components for maintainability
- **Accessibility**: SEO-friendly structure and accessibility support
- **Performance**: Optimized for fast loading and smooth interactions

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 3
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Package Manager**: npm

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone or navigate to the project directory:
```bash
cd obscura-landing
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📦 Building for Production

```bash
npm run build
npm start
```

## 🎨 Customization

### Colors
Edit the color palette in `tailwind.config.js`:
- `accent-gold`: Primary accent color (#d4af37)
- `accent-warm`: Warm accent color (#c97f5c)
- `accent-cream`: Light text color (#f5f1e8)
- `dark-*`: Background gradients

### Content
Update content in individual section components located in `components/sections/`:
- Modify text, images, and prices
- Change testimonials and FAQs
- Update location information

### Animations
Adjust Framer Motion animations in each component:
- `variants` for staggered animations
- `whileHover` and `whileTap` for interactions
- `animate` for continuous animations

## 🔧 Configuration

### Environment Variables
Create a `.env.local` file for environment-specific settings:
```
NEXT_PUBLIC_API_URL=your_api_url
```
